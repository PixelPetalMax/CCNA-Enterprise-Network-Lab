# Router-on-a-STick VLAN Gateway
#Overview
Implement VLAN segmentation on a Layer 2 switch and provide default gateway functionality using router-on-a-stick architecture.

## Technologies Used
-Cisco Packet Tracer
-802.1Q VLAN trunking
-Sub-interfaces
-IPV4 addressing

## Network Design
-Vlan 10:Users
-Vlan 20: Servers
-Vlan 30: IT
-Single router providing inter-VLAN gateway services

## Validation
-Successful ICMP from hosts to VLAN gateways
-Verified trunking and VLAN assignments
-Packet flow inspected in Simulation Mode

## Skills Demonstrated
-VLAN configuration
- Trunking
- Router sub-interfaces
- -Network troubleshooting fundamentals
