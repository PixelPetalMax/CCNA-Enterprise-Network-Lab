# CCNA-Enterprise-Network-Lab
#Overview
Simulated enterprise network covering VLANS, OSPF, ACLs, NAT, and HSRP. Includes packet analysis and troubleshooting.

## Technologies Used
-VLANS
-Router-on-a-Stick
-OSPF
-ACLs
-NAT
-HSRP
-DHCP & DNS

## Validation
-Verified inter-VLAN routing
-Observed OSPF convergence
-Tested ACL and NAT functionality
-HSRP failover simulations

## Key Takeaways
-Hands-on CCNA-compliant routing and switching experience
-Troubleshooting network connectivity 
-Network monitoring and redundancy verification.
